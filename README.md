# USTConvert
USTConvert：Convert UTF-8 UST to Shift_JIS for UTAU

![1](1.png)

## 使用方法

1.先用UTAU打开要转换的UST

![2](2.png)

2.点击另存为，保存为UST形式

![3](3.png)

3.右键UST，用记事本打开

![4](4.png)

4. 点击另存为

![5](5.png)

5.编码选择UTF-8

![6](6.png)

6.点击替换（也可以保存为不同文件名）

![7](7.png)

7.打开转换器并转换UST

![8](8.png)

![10](10.png)

8.保存转换后的UST

![11](11.png)
